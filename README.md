# Help Desk Runbooks

Ticket-style IT support runbooks and homelab notes focused on Tier 1 troubleshooting.
Written to be reproducible: symptoms → checks → fix → validation → prevention.

## Structure
- `runbooks/` – categorized runbooks (Windows, networking, Linux, etc.)
- `templates/` – reusable runbook templates
- `assets/images/` – screenshots and handwritten summaries

## Runbook format
Each runbook includes:
- **Issue / Symptoms**
- **Environment**
- **Quick checks**
- **Step-by-step fix**
- **Validation**
- **Notes / prevention**
- **References** (official docs)
